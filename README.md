## 环境说明

运行环境：Python3（已测试环境Python3.7.4）
Python依赖：playsound （使用pip命令安装）

## 使用方法

运行main.py，在控制台中提示输入序列长度。此时输入要记忆的数字序列的长度，如`5`。
然后程序将随机生成设定序列长度的一段语音。仅播放一遍。
此时请认真听语音。语音结束后，回忆语音所播报的数字序列。
将回忆起的数字序列输入控制台，如`52901`，回车提交。
控制台将输出语音播放的数字序列和你输入的数字序列，并判断二者是否一致。
最后控制台询问是否继续测试，直接回车或输入Y/y后回车，将继续测试。输入n或其他字符后回车，停止测试并退出程序。

认知科学实验数据为，正常人对数字/字母的记忆长度为7±2。
